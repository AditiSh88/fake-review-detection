# Fake Review Detection Using Machine Learning

## Overview
Fake and manipulated online reviews are a major problem in e-commerce platforms. This project uses Machine Learning techniques to automatically classify reviews as Genuine or Fake based on textual features.

## Tech Stack
- **Language**: Python
- **Libraries**: Scikit-learn, Pandas, NumPy, NLTK
- **Model**: Logistic Regression, Naive Bayes
- **Frontend**: Streamlit
